## Installing packages

- Windows:
    ```cmd
    gpi_install.bat
    ```
- Linux:
    ```bash
    ./gpi_install.sh
    ```

## 1st terminal: website launch

- Windows:
    ```cmd
    gpi_run_frontend.bat
    ```
- Linux:
    ```bash
    ./gpi_run_frontend.sh
    ```

## 2nd terminal: starting the server API

- Windows:
    ```cmd
    gpi_run_backend.bat
    ```
- Linux:
    ```bash
    ./gpi_run_backend.sh
    ```

## Links

- http://localhost:3000/ - Frontend React App
- http://localhost:3001/ - Backend Express server
